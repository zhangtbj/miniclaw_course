# 待办任务提取

从一段自然语言文本中，识别并结构化提取所有待办任务。

## 处理步骤

1. **识别任务**：找出文本中所有「需要有人去做」的事项，过滤掉纯信息性陈述。
2. **补全字段**：
   - `owner`：负责人，文本未提及则填「待确认」；
   - `task`：用一句话描述清楚要做什么；
   - `deadline`：截止时间，文本未提及则填「待确认」。
3. **去重**：相同任务只保留一条。
4. **语言**：所有输出使用中文。

## 输出要求

返回结构化结果，包含一个 `tasks` 字段，它是对象列表，每个对象含：

- `owner`：负责人
- `task`：任务描述
- `deadline`：截止时间

不要输出额外解释。
